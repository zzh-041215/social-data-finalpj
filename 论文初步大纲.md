**主标题：**
《"结婚买房还是躺平"：B站人生规划话语中的解构性内容为何传播更广？》

**备选（学术味更重）：**
《优绩主义的反叙事如何扩散？——基于B站人生规划议题视频的文本与传播分析》

## 一、引言（约500字）

### 1.1 现象观察与研究动机

中国互联网上存在一套相对稳定的"标准人生剧本"：毕业→就业→买房→结婚→生育，并附带一套配套叙事——努力会有回报、教育能带来阶层流动、个人奋斗终将变现。然而近年来，这套叙事在B站（Bilibili）的人生规划类内容中遭受大规模解构：讨论"要不要买房"的视频评论区充斥着"韭菜就是用来割的""彩礼本质是什么""生孩子是传递痛苦"这类带有明显结构性批判色彩的表达；"孔乙己文学"嘲讽高学历低就业的吊诡处境；"不婚不育保平安"成为部分群体的自我保护宣言。

这种话语风格不是简单的"负面情绪"，而是一种更复杂的认知操作：它承认现实的结构性约束，用反讽和解构来拒绝原有意义体系，同时又不构成明确的政治主张。本文将其命名为"结构性批判话语"（structural-critique discourse），与情绪宣泄型的焦虑话语区分开来。

核心观察是：这类内容在B站似乎具有异常强的传播能力——互动率高、评论区讨论激烈。这引发了一个可测量的研究问题：在控制视频基础属性后，结构性批判话语的传播效果是否系统性地优于其他情绪类型（积极叙事、焦虑表达、中性信息）？

### 1.2 三个核心研究问题

- **RQ1（描述）**：B站人生规划议题视频中，各情绪类型（积极/传统认同、焦虑/压力表达、结构性批判、中性信息）的分布情况如何？是否随时间推移发生了系统性变化？
- **RQ2（关联）**：控制视频基础属性后，情绪类型与传播效果（综合互动率）是否存在统计显著的关联——尤其是结构性批判话语是否具有传播优势？
- **RQ3（机制讨论）**：若传播优势存在，它更可能来自内容本身的共鸣效应、平台算法的正反馈循环，还是创作者的风格模仿？本文能在多大程度上区分这三种解释？

### 1.3 研究贡献

将NLP文本分析方法应用于中国青年话语研究，提供长时间序列的量化证据；同时正面讨论因果识别困难，探讨社会科学数据科学方法的边界。

## 二、文献综述（约700字）

### 2.1 情绪传染与在线传播

Kramer等人（2014）在Facebook上的自然实验表明，接触到更多负面内容的用户，自身发布的内容也更趋于负面，提供了在线情绪传染的因果证据。Berger（2011）的研究则指出，在新闻内容中，引发高唤醒情绪（包括焦虑、愤怒，以及敬畏等积极情绪）的内容扩散率更高，而低唤醒的悲伤情绪扩散率反而较低。这为研究情绪类型与传播效果的关系提供了理论框架。

### 2.2 优绩主义及其批判

Sandel（2020）在《优绩的暴政》中指出，优绩主义（meritocracy）在制造成功者"应得"感的同时，也制造了失败者的"活该"感，将结构性不平等内化为个人道德问题。这一批判在中国语境下有其特殊共鸣：高考制度、学历崇拜与"努力改变命运"的叙事长期共存，但当阶层流动通道收窄（房价上涨、就业市场收紧、教育内卷），优绩主义叙事就面临合法性危机。

国内研究中，关于"孔乙己文学"（2023年）、"彩礼争议"、"丁克/不婚选择"的话语分析已有一定积累，但多为质性解读，缺乏对这类话语传播机制的量化考察。本文试图在这一方向上补充量化证据。

### 2.3 "人生剧本"话语与青年身份协商

社会学家将青年期的关键决策（学业、就业、婚育、置业）视为身份认同建构的核心场域。在中国，"结婚-买房-生育"构成一套隐性的标准化时间表（normalized life schedule），对不符合这一时间表的青年施加社会压力。

近年来B站出现大量对这一时间表进行再审视甚至解构的内容：从"买房到底值不值"的理性算账，到"催婚催育是剥削"的批判性分析，再到"彩礼刺客"等新词的流行，构成了一场去中心化的集体话语协商。B站的弹幕与评论区为这种协商提供了实时的公共讨论空间，也为本文提供了研究文本。

### 2.4 B站作为研究场域

B站月活用户中18–35岁比例长期超过80%，且内容评论体系（弹幕、评论区）为情绪分析提供了丰富的文本信号。已有研究将B站用于二次元文化、饭圈话语、知识传播等方向，但针对人生规划议题情绪话语的系统量化研究尚属空白。

### 2.5 中文情绪文本分析方法

现有方法分为两大类：
- **词典法**：基于情感词典（如大连理工情感词汇本体、NTUSD）进行词频统计，优点是透明可解释，适合自定义情绪类别。
- **预训练模型**：基于BERT/RoBERTa-Chinese的fine-tuning方法在标准中文情感分析任务上显著优于词典法，但需要标注训练数据。

### 2.6 研究空白

现有关于在线情绪传播的研究多集中于欧美平台（Twitter、Facebook），且多为横截面设计。优绩主义批判作为一种具体话语类型的传播效果，在中文社交媒体上尚未获得量化研究的关注。本文试图填补这一空白。

## 三、数据与研究设计（约600字）

### 3.1 平台与内容范围界定

**平台**：B站（Bilibili）

选择依据：用户年轻化；内容多样且存在公开互动数据（播放量、点赞、投币、收藏、评论）；弹幕和评论区提供丰富文本信号；人生规划类话题在此平台高度活跃。

**内容范围：关键词搜索，而非固定分区**

本研究不限定于某一固定分区（B站的分区划分与话题边界不完全吻合），而是通过关键词搜索覆盖目标议题：

| 议题 | 搜索关键词 |
|------|-----------|
| 婚育决策 | 要不要生孩子、不婚不育、催婚、彩礼、丁克 |
| 置业决策 | 要不要买房、租房还是买房、房价、首付 |
| 职业与阶层 | 孔乙己、考研值不值、考公、学历贬值、阶层固化 |
| 综合人生规划 | 人生规划、躺平、人生剧本、二十几岁怎么活 |

> **说明**：通过关键词搜索而非分区抓取，能更精准命中目标话题，同时覆盖了"职场区""生活区""知识区"等多个分区中的相关内容。

### 3.2 数据收集

**方法**：B站公开API（视频信息接口）＋ 自动化爬取（合规说明：仅采集公开信息，不收集用户个人识别数据，采集频率控制在平台限制范围内）

**数据字段**：

| 字段 | 说明 |
|------|------|
| 视频标题 | 用于情绪分类 |
| 视频简介 | 辅助分类 |
| 发布时间 | 时间序列分析的核心变量 |
| UP主粉丝数 | 控制变量（创作者规模） |
| 播放量 | 控制变量/传播基础 |
| 点赞、投币、收藏、评论数 | 计算互动率 |
| 评论文本（随机抽样，每视频20条） | 辅助情绪分析 |

**时间范围**：2020年1月—2025年12月（以季度为基本分析单位）

**样本规模（目标）**：每季度每分区约500–1000个视频，合计约15,000–30,000个视频；评论抽样约30万条

### 3.3 数据清洗

- 剔除播放量低于1,000的视频（过低曝光量，传播指标失真）
- 剔除明显广告内容（标题含"广告""合作"等关键词）
- 评论去重，剔除纯表情/空内容评论

## 四、研究方法（约500字）

### 4.1 情绪分类框架（四类）

| 情绪类型 | 操作化定义 | 核心词汇示例 |
|---------|-----------|------------|
| **积极/理想主义** | 正向价值导向、目标感、热情 | 热爱、梦想、奋斗、努力、成长、相信、未来 |
| **焦虑/消极** | 明确的负面情绪表达 | 焦虑、崩溃、迷茫、压力、害怕、失业、卷不动 |
| **解构/犬儒** | 反讽、自嘲、戏谑、消解意义 | 摆烂、整活、发癫、典、绷不住、抽象、赢麻了、小丑 |
| **中性/信息型** | 信息陈述、客观描述，无明显情绪倾向 | — |

### 4.2 情绪识别方法

**主要方法（词典法，适合课程论文）**：

1. 基于已有中文情感词典扩充，构建上述四类词典
2. 对视频标题（主要）＋简介（辅助）进行加权词频统计
3. 按最高分类情绪赋予类别标签

**验证**：随机抽取500条标注与人工标注对比（报告Cohen's Kappa值）

**进阶选项（若条件允许）**：调用已有中文情感分析API（如百度NLP / HanLP）作为对比验证基线。

### 4.3 传播效果指标

构建综合互动率（Engagement Rate）：

```
互动率 = (点赞数 + 投币数 + 收藏数 + 评论数) / 播放量
```

这一指标相比原始互动量，控制了视频曝光量的差异，更能反映内容本身对受众的吸引力。

### 4.4 统计模型

**核心回归（OLS）**：

```
log(互动率ᵢ) = β₀ + β₁·情绪类型ᵢ + β₂·时间趋势ᵢ + β₃·log(粉丝数ᵢ) 
               + β₄·视频时长ᵢ + β₅·发布时点ᵢ + εᵢ
```

- 情绪类型为四类虚拟变量，以"中性"为参照组
- 时间趋势：年份或季度固定效应
- 对因变量取对数以处理偏态分布
- 报告稳健标准误

**辅助分析（RQ1）**：各季度四类情绪占比的折线图，展示演化趋势。

## 五、预期结果与分析（约800字）

### 5.1 情绪演化趋势（时间序列，对应RQ1）

**预期展示形式**：折线图，x轴为季度，y轴为各情绪类型占所有视频的比例。

**预期核心发现（假设）**：
- 积极/理想主义情绪占比整体下降，在2022年（就业市场压力峰值年）出现显著节点
- 解构/犬儒情绪占比持续上升，2021年"躺平"话语爆发后明显加速
- 焦虑情绪占比呈现波动而非单调上升（可能有舆论周期性）

**关键历史节点分析**（结合外部事件作为描述性解读）：

| 时间节点 | 外部事件 | 预期情绪影响 |
|---------|---------|------------|
| 2021年下半年 | 教育"双减"政策、平台整治 | 焦虑情绪上升 |
| 2022年 | 疫情管控高峰、应届生就业率下降 | 解构情绪爆发 |
| 2023年 | 经济复苏叙事与现实落差 | 犬儒情绪持续 |
| 2024–2025年 | AI浪潮引发就业替代讨论 | 焦虑/解构混合上升 |

### 5.2 情绪类型与传播效果（回归结果，对应RQ2）

**预期展示形式**：OLS回归系数表（含置信区间和显著性标记）

**预期核心发现（假设）**：
- 解构/犬儒类内容的互动率β系数显著为正（相对于中性基准）
- 焦虑类内容互动率也偏高，但幅度小于解构类
- 积极/理想主义内容与中性内容无显著差异，或甚至低于基准

若假设成立，则在描述层面支持"解构性内容传播效果更好"的观察。

### 5.3 词汇层面的微观证据（补充分析）

选取10–15个标志性词汇，绘制各词汇在标题中出现频率的年度变化曲线：

**上升词汇组**："摆烂""内卷""整活""发癫""绷不住"  
**下降词汇组**："梦想""热爱""奋斗""相信""未来"

这组对比图视觉冲击力强，能直观佐证整体趋势，适合放在结果部分开头。

## 六、讨论（约700字）

### 6.1 对研究问题的回答

综合上述发现，简要回应三个RQ。特别说明：本文可以回答RQ1（描述趋势）和RQ2（关联性），但**不能直接回答**"解构情绪导致传播"这一因果命题。

### 6.2 关联背后的三种解释机制

这是论文最有深度的部分，展示数据科学与社会科学交叉的思维：

**机制一：共鸣效应（内容本身）**  
解构性内容击中了青年群体的真实情绪状态，产生"说出了我想说的话"的认同感，因而评论互动更多。这是内容驱动的解释。

**机制二：算法正反馈（平台机制）**  
如果平台推荐算法以互动率为优化目标，而解构内容天然互动率更高，则算法会优先推送此类内容，形成正反馈循环，进一步推高解构内容的占比和互动量。内容的情绪分布可能是平台算法与用户偏好共同演化的结果，而非单纯反映社会情绪。

**机制三：创作者策略模仿（供给侧）**  
UP主观察到解构风格内容流量更高，主动迁移创作风格。这使得因果方向变为：互动反馈 → 情绪话语分布，而非情绪 → 传播效果。

**关键判断**：这三种机制并不互斥，很可能同时存在。本文的OLS回归无法在它们之间进行识别。

### 6.3 因果识别的困难——与课程方法的对话

这是本文最重要的方法论反思，也是与课程核心内容（Angrist & Pischke, 2009）直接对话的部分：

**核心问题**：存在明显的遗漏变量偏误（OVB）。宏观社会事件（如就业压力）可能同时影响：①创作者的情绪倾向（自变量），②用户的共鸣强度（因变量），导致β₁系数的估计是有偏的。

**未来可能的改进方向**（展望，而非本文实现）：

- **事件研究法（Event Study）**：以某次平台算法重大更新或政策冲击为外生变化，观察特定时间窗口内情绪分布和互动率的断点变化。
- **双重差分法（DiD）**：对比受某一事件影响较大的话题分区（如求职区）与受影响较小的分区（如游戏区）的情绪变化轨迹。如果两者差异在事件前不存在、事件后出现，则可作为因果效应的间接证据。
- **控制创作者固定效应**：在现有数据框架内，控制UP主个体固定效应，可以排除部分"本来就倾向犬儒的创作者被推荐更多"这一混淆。

### 6.4 研究局限

- 数据仅覆盖B站特定分区，不能代表全体中国青年网络行为
- 词典法情绪分类存在准确率上限，尤其是对反讽语境的识别较弱
- 平台算法的实际运作机制不透明，无法直接建模

## 七、结论（约400字）

### 7.1 主要发现总结

- 是否观察到情绪演化趋势（时间维度）
- 解构/犬儒内容是否具有更高传播效果（截面维度）
- 关联背后机制的多元性

### 7.2 理论与方法层面的贡献

- 将情绪演化研究从质性观察推向量化验证
- 展示因果推断框架在数字媒体研究中的应用边界
- 为B站作为社会情绪观测场域提供方法论参考

### 7.3 对平台治理的启示

若解构内容的传播优势部分由算法驱动，则平台推荐机制的设计可能在无意中放大了犬儒情绪的扩散。这对于理解青年心理健康的数字环境具有政策意义。

### 7.4 未来研究方向

- 纳入弹幕数据（B站独有的实时评论形式）
- 引入双重差分或事件研究法以强化因果识别
- 跨平台对比（B站 vs 知乎 vs 小红书）
