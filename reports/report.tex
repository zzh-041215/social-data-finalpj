\begin{abstract}
\noindent
近年来，中国互联网上“毕业—就业—买房—结婚—生育”的标准人生剧本在知乎人生规划类讨论中遭到大规模解构。本文采集知乎四类人生规划议题(婚育、置业、职业与阶层、综合人生规划)的 2503 条回答与 1.35 万条评论(2020年一季度--2025年四季度)，
用基于词典的四类中文情绪分类器(积极、焦虑、解构、中性；人工校验 Cohen's Kappa $=0.78$)刻画情绪分布的时间演化，并用 OLS 回归、事件研究与双重差分检验情绪类型与传播效果的关联。

结果表明，解构话语集中于综合人生规划与职业议题，并于2023Q1前后达到占比高点；控制固定效应后，三类情绪话语获赞均显著低于中性话语；解构话语在情绪话语内部点估计最高，但差异未达统计显著。
\par\medskip
\noindent\textbf{关键词:}优绩主义批判；解构话语；情绪分类；在线传播；知乎
\end{abstract}

\section{引言}

中国互联网上长期存在一套相对稳定的人生剧本:毕业$\rightarrow$就业$\rightarrow$买房$\rightarrow$结婚$\rightarrow$生育，并附带一套配套叙事——努力会有回报、教育能带来阶层流动、个人奋斗终将变现。
然而近年来，这套叙事在知乎的人生规划类讨论中遭受大规模解构:例如经典的是否买房的问题下，部分高赞回答转换为“韭菜就是用来割的”、“彩礼本质是什么”这类带有结构性批判色彩的表达；
“孔乙己文学”用来嘲讽高学历低就业的尴尬处境；“不婚不育、丁克、低消费”成为一部分群体为了避免结构性压迫的自我保护之策。

这种话语一方面确实反映了一代人对于经济下行、生活成本升高的负面情绪宣泄。
不过在另一方面，它承认现实的结构性约束，用反讽与解构来消解原有意义体系，但同时有没有能够建立起新的政治主张。
本文借用 Sandel(2020)对优绩主义(meritocracy)的批判，将其理解为对"优绩叙事"的去合法化反叙事。
我们发现这类内容在知乎似乎具有较强的传播能力，这就可以建模为一个数据科学问题——在控制回答基础属性后，解构话语的传播效果是否系统性地优于其他情绪类型？

本文提出三个研究问题:
\begin{enumerate}[leftmargin=2.4em,itemsep=2pt]
  \item \textbf{描述}各情绪类型(积极、焦虑、解构、中性)在知乎人生规划议题回答中的分布如何？是否随时间发生系统性变化？
  \item \textbf{关联}控制回答基础属性后，情绪类型与传播效果是否存在显著关联，尤其解构话语是否具有传播优势？
  \item \textbf{机制}若关联存在，它更可能源于内容共鸣、算法正反馈还是创作者模仿？本文能在多大程度上区分这些解释？
\end{enumerate}

本文贡献在于，将 NLP 文本分析方法应用于中国青年话语研究，提供长时间序列的量化证据；同时正面讨论因果识别的困难，探讨社会科学数据科学方法的边界。

\section{文献综述}

\subsection{情绪传染与在线传播}
Kramer 等人(2014)在 Facebook 上的自然实验表明，接触更多负面内容的用户，其自身发布的内容也更趋负面，提供了在线情绪传染的因果证据。
Berger(2011)进一步指出，引发如焦虑、愤怒以及敬畏等“高唤醒情绪”的内容扩散率更高，而低唤醒的悲伤情绪扩散率反而较低。
Brady 等人(2017)则发现道德化情绪词显著提升内容在社交网络中的扩散。这为研究情绪类型与传播效果的关系提供了理论框架，也提示我们不同情绪在传播上并不等价。

\subsection{优绩主义及其批判}
Sandel(2020)在《优绩的暴政》中指出，优绩主义既使成功者将成功完全归功与自身的努力和付出，同时也让失败者心甘情愿承认自身能力的有限而居于人下，将结构性不平等内化为个人道德问题。
这一批判在中国语境下有特殊共鸣:高考制度、学历崇拜与优绩注意叙事长期共存，但当阶层流动通道收窄(房价上涨、就业收紧、教育内卷)，优绩叙事便面临合法性危机。
国内关于孔乙己文学、彩礼争议、丁克/不婚选择的话语分析已有积累，但多为质性解读，缺乏对其传播机制的量化考察。

\subsection{青年身份与传统叙事协商}
社会学将青年期的关键决策(学业、就业、婚育、置业)视为身份建构的核心场域。
在中国，“结婚—买房—生育”构成一套隐性的标准化时间表，对不符合者施加社会压力。然而以青年用户为主的知乎，近年出现大量对该时间表进行再审视乃至解构的内容，展现出一种自发性对传统价值观念的反叛。

\subsection{知乎作为研究场域与中文情绪分析方法}
知乎用户以城市青年知识阶层为主，问答结构天然适合话语分析:问题设定议题框架，回答构成话语文本，赞同机制提供清晰的互动信号；长文本特性也使基于词典的中文情绪分类更可靠。
中文情绪分析方法主要分为\textbf{词典法}(如大连理工情感词汇本体，透明可解释、便于自定义类别)与\textbf{预训练模型}(BERT/RoBERTa-Chinese，精度高但需标注数据)。
综合可解释性与课程论文的条件约束，本文采用词典法。现有在线情绪传播研究多集中于欧美平台且多为横截面设计，优绩主义批判作为一种具体话语类型的传播效果，在中文社交媒体上尚未获得量化关注，本文试图填补这一空白。

\section{数据与研究设计}

\subsection{平台与内容范围}
本研究以知乎为平台，通过关键词搜索覆盖四个人生规划议题领域:婚育决策(不婚不育、彩礼、生育成本等)、置业决策(房价太高、要不要买房、房贷压力等)、职业与阶层(孔乙己、学历贬值、内卷、35 岁危机等)与综合人生规划(躺平、摆烂、精神内耗、年龄焦虑等)。
关键词搜索相比话题广场抓取能更精准命中目标内容。

\subsection{数据收集与清洗}
通过知乎公开搜索与回答接口自动化采集公开信息。
采集字段包括:问题标题、回答内容(情绪分类主输入)、发布时间、答主粉丝数、赞同/评论/感谢数，以及每条回答随机抽样的评论文本。
时间范围 2020 年 1 月--2025 年 12 月，以季度为基本分析单位。

爬取得到数据之后，我们进一步将剔除回答长度低于 50 字的内容和明显的广告内容；评论去重并剔除纯表情/空内容。
清洗后得到 \textbf{2503 条回答}与 \textbf{13{,}529 条评论}。值得注意的是，搜索接口偏好返回近期热门回答，样本明显偏向 2024--2025 年(2025Q4 共 385 条，2020Q1 仅 25 条)。

\section{研究方法}

\subsection{情绪分类框架}
本文将回答情绪操作化为四类:\textbf{积极/理想主义}(热爱、梦想、奋斗、成长)、\textbf{焦虑/消极}(焦虑、崩溃、压力、失业)、\textbf{解构/犬儒}(摆烂、整活、发癫、典、赢麻了)与\textbf{中性/信息型}(客观陈述，无明显情绪倾向)。在大连理工情感词汇本体基础上扩充，构建四类词典(积极 91 词、焦虑 99 词、解构 142 词)。

\subsection{分类流程与验证}
使用 jieba 分词后对回答内容进行加权词频统计，并引入否定词与程度副词规则修正情绪强度，按情绪密度最高者赋予类别标签；当所有情绪词密度低于阈值时判为中性。设第 $k$ 类的加权得分为 $s_k$，则类别标签为
\begin{equation}
\hat{y}_i = \operatorname*{arg\,max}_{k\in\{\mathrm{pos},\,\mathrm{anx},\,\mathrm{dec}\}} s_{ik},
\qquad \hat{y}_i = \mathrm{neu}\ \ \text{if}\ \ \max_{k}\frac{s_{ik}}{L_i} < \tau,
\end{equation}
其中 $L_i$ 为回答词数，$\tau$ 为中性密度阈值。人工标注 499 条与模型对比，采用 Cohen's Kappa 衡量一致性:
\begin{equation}
\kappa = \frac{p_o - p_e}{1 - p_e},
\end{equation}
其中 $p_o$ 为观测一致率、$p_e$ 为期望一致率。结果 $\kappa = 0.78$(substantial)，准确率 $0.84$，人工中性占比 $33.3\%$ 与模型 $35.1\%$ 高度吻合。
最主要的混淆点来自于积极/解构和中性之间的互相混淆，因为反讽长文容易被判别为中性，这确实符合词典法的已知局限。

\subsection{传播效果回归模型}
主回归为 OLS，以中性为参照组:
\begin{equation}
\begin{split}
\log(\mathrm{approval}_i + 1) ={}& \beta_0 + \sum_{k\in\{\mathrm{pos},\,\mathrm{anx},\,\mathrm{dec}\}}\beta_k\,\mathbb{1}[\mathrm{emotion}_i = k] + \gamma_1 \log(\mathrm{fans}_i + 1) \\
&+ \gamma_2 \log L_i + \gamma_3\,\mathrm{age}_i + \delta_{q(i)} + \theta_{g(i)} + \varepsilon_i,
\end{split}
\label{eq:ols}
\end{equation}
其中 $\delta_q$ 为季度(quarter)固定效应、$\theta_g$ 为议题(topic)固定效应，使用 HC1 异方差稳健标准误。
为检验情绪话语内部差异，进一步对 $\beta_{\mathrm{dec}}-\beta_{\mathrm{pos}}$、$\beta_{\mathrm{dec}}-\beta_{\mathrm{anx}}$ 做线性约束检验。

\subsection{准实验设计}
因为 2023 年前后的数据样本数量均可观，且第一季度出现了爆火全网的“孔乙己文学”进一步冲击了就业/学历叙事，本文以关键词 $j$、季度 $t$ 为单元构造面板。
设处理组为职业与阶层议题、对照组为婚育$+$置业议题，估计双重差分:
\begin{equation}
Y_{jt} = \alpha + \rho\,(\mathrm{Treat}_j \times \mathrm{Post}_t) + \mu_j + \lambda_t + u_{jt},
\label{eq:did}
\end{equation}
$Y_{jt}$ 分别取解构话语占比与平均 $\log$ 赞同，标准误按关键词聚类。事件研究版本将 $\mathrm{Treat}_j\times\mathrm{Post}_t$ 替换为各季度的交互项以检验平行趋势。
此外估计加入作者固定效应的模型，即仅含发文不少于两篇的作者，以排除创作者情绪影响被推荐内容这个混淆。

\section{结果与分析}

\subsection{情绪分布与时间演化}

总体分布为:中性 $35.5\%$、积极 $25.2\%$、解构 $19.9\%$、焦虑 $19.5\%$(图~\ref{fig:dist})。分议题看，解构话语高度集中于\textbf{综合人生规划}(约 $60\%$)与\textbf{职业与阶层}(约 $32\%$)，而婚育、置业议题以中性$+$焦虑为主——这表明解构话语是建立在人生轨迹的话题之上的。

  \includegraphics[width=0.9\textwidth]{emotion_dist.png}
  \caption{四类情绪的总体分布($N=2503$)。}

季度演化(图~\ref{fig:trend})显示，解构话语在 \textbf{2023Q1} 出现占比高点($\approx 0.48$)，与“孔乙己文学”舆论事件的时点吻合；
中性/信息型话语在 2024--2025 年抬升至约 $0.38$。
不过值得注意的是，由于搜索接口偏向返回近期存量热门回答，时间序列反映的是当下仍然可见的存量话语，无法真正获得各个时间点的全量发文，存在幸存者偏差。

主模型(式~\ref{eq:ols}，$N=2503$，$R^2=0.232$)结果见表~\ref{tab:ols}。控制变量符合预期:$\log$ 粉丝数($\gamma_1 = 0.226$)与 $\log$ 长度($\gamma_2 = 0.408$)均显著为正。
主要结果有两层:

我们可以看到三类情绪话语的获赞均显著低于中性/信息型话语($p<0.001$)。
中性类多为长篇理性分析，这种分析性回答在知乎获赞最高——这与情绪化内容更易传播的直觉相反，
但在四个稳健性设定(综合互动量、议题$\times$季度交互、高赞子样本)下符号一致，是稳健且可写的发现。

不过在情绪话语内部，解构类点估计最高($\beta_{\mathrm{dec}}=-0.47 > \beta_{\mathrm{anx}}=-0.64 > \beta_{\mathrm{pos}}=-0.66$)，方向上支持解构类传播效果最优的假说。
但直接对比检验显示，解构$-$积极 $=+0.19$($p=0.14$)、解构$-$焦虑 $=+0.17$($p=0.22$)，未达统计显著。阈值敏感性分析表明，解构$-$积极的差异仅在中性占比 $\le 21\%$ 时才显著，而这些阈值对应的人工 Kappa 更低($0.61$--$0.66$)，故本文\textbf{不主张}该显著性。

综上，本文发现，在知乎这个社交媒体平台中，信息型话语获赞相对最高，而情绪化的话语获赞普遍比较低。在情绪话语内部观察到\textbf{解构话语}有相对较优的传播倾向，不过值得注意的是，这个差异并没有达到$p=0.05$的统计学显著性。

\subsection{词汇层面的微观证据}

为在词汇粒度上佐证上述情绪演化，本文选取两组标志性词汇追踪其逐年出现率:\textbf{解构上升组}(躺平、摆烂、内卷、整活、绷不住)与\textbf{优绩下降组}(梦想、奋斗、相信、热爱、未来)，结果见图~\ref{fig:marker}。两组曲线呈现清晰的剪刀差:上升组词频自 2021 年"躺平"话语爆发后持续走高，并在 2022--2023 年职业焦虑高峰期进一步抬升；下降组词频则总体走平甚至缓降，昔日高频的奋斗叙事用词在样本中逐渐被稀释。

\section{讨论}

\subsection{对研究问题的回答}
本文回答了问题1与问题2，并部分讨论问题3。结果表明解构话语呈现阶段性扩张，但其传播优势主要表现为方向性而非统计显著优势。对于问题3，由于数据与识别条件限制，本文仅能提供关联层面的解释。

\subsection{关联背后的机制}
本文观察到的关联可以有几种机制解释。首先是共鸣效应，互联网上的解构内容击中青年真实情绪，能够增加认同感的同时暂时消解在社会中遇到的痛苦，从而获得更多赞同；而在获得情绪共鸣之外，中青年仍希望理性的人生问题能够通过互联网的交流得到解决，因此中性表达的受欢迎程度要高于其余三者。这是内容驱动的解释，与作者固定效应下关联仍存在相一致。
其次是算法驱动的正反馈:若平台推荐以互动为优化目标，而某类内容天然互动更高，算法会优先推送，形成正反馈，使情绪分布成为算法与用户偏好共同演化的结果。
此外还需要考虑的是创作者模仿。创作者观察到某种风格更受欢迎而主动迁移风格，让互动反馈量反作用于话语情绪的分布。

\subsection{研究局限}
本文的局限大致可归为三类。首先是传播指标的退化，由于接口已不再下发浏览量，本文只能以对数赞同数叠加曝光代理变量来近似传播效果，而非严格意义上的互动率。
其次是分类方法的边界，词典法对反讽语境识别较弱，尽管整体一致性达到 $\kappa=0.78$，解构话语仍易与中性互混，从而可能低估解构效应。
最后是因果设计的功效不足，议题级聚类数偏少、处理与对照的划分基于研究者判断、早期单元样本稀疏，共同导致 DiD 的置信区间偏宽、统计功效有限。

%==================================================================
\section{结论}

本文基于知乎四类人生规划议题的 2503 条回答，量化刻画了优绩主义反叙事(解构话语)的分布、演化与传播效果。主要发现有三:其一，解构话语集中依附于综合人生规划与职业议题，并在 2023Q1“孔乙己文学”前后出现可见高点；
其二，控制曝光与固定效应后，信息型理性话语获赞最高、情绪化话语普遍较低，与情绪驱动传播的直觉相反；
其三，解构话语在情绪话语内部呈现方向性的传播优势，但未达统计显著，且事件研究与 DiD 未能提供因果证据。

在方法层面，本文展示了将情绪演化研究从质性观察推向量化验证的可行路径，也清晰呈现了因果推断框架在数字媒体研究中的应用边界——OVB、幸存者偏差与平台算法不透明共同构成识别障碍。在政策层面，若解构内容的传播优势部分由算法驱动，则推荐机制可能在无意中放大犬儒情绪的扩散，对理解青年心理健康的数字环境具有启示。
未来研究可引入跨平台对比(知乎 vs B 站 vs 小红书)、寻找更干净的外生冲击以强化因果识别，并以预训练模型提升对反讽语境的识别精度。
