# 知乎人生规划话语数据需求说明

## 研究背景

论文题目：《优绩主义的反叙事如何扩散？——基于知乎人生规划议题回答的文本与传播分析》

核心研究问题：知乎上关于"结婚买房还是躺平"等人生规划话题的回答中，不同类型的情绪话语（积极/焦虑/解构/中性）如何随时间演化？哪种情绪类型传播更广？

---

## 需要的数据

### 1. 回答元数据（answers表）

| 字段 | 说明 | 用途 |
|------|------|------|
| `answer_id` | 知乎回答ID（如 `239847561`） | 唯一标识 |
| `question_id` | 知乎问题ID | 关联问题上下文 |
| `question_title` | 问题标题 | **语境框架（辅助情绪分类）** |
| `content` | 回答全文 | **情绪分类主输入** |
| `excerpt` | 回答摘要 | 辅助快速分类 |
| `publish_time` | 发布时间（精确到日） | **时间序列分析核心** |
| `author_name` | 答主名称 | — |
| `author_url_token` | 答主唯一标识 | 关联用户数据 |
| `author_follower_count` | 答主粉丝数 | 控制变量（创作者规模） |
| `voteup_count` | 赞同数 | **互动率分子** |
| `comment_count` | 评论数 | 互动率分子 + 描述统计 |
| `view_count` | 浏览量 | **互动率分母**（视API返回情况，若不可用则以赞同数为曝光代理） |
| `favorite_count` | 收藏数 | 互动率分子（视API返回情况） |
| `keyword_searched` | 命中关键词 | 来源追踪 |
| `topic` | 所属议题（婚育/置业/职业/综合） | 分类分析 |

### 2. 评论文本（comments表）

| 字段 | 说明 | 用途 |
|------|------|------|
| `comment_id` | 评论ID | 唯一标识 |
| `answer_id` | 所属回答ID | **关联answers表** |
| `question_id` | 所属问题ID | 辅助关联 |
| `content` | 评论文本内容 | 情绪分析补充输入 |
| `like_count` | 评论点赞数 | 评论热度 |
| `publish_time` | 评论时间 | — |
| `parent_id` | 父评论ID（0=一级评论） | 过滤楼中楼 |

## 技术说明（供参考）

- 数据来源：知乎公开API（`www.zhihu.com/api/v4`）
- 搜索接口：`/api/v4/search_v3?q=KEYWORD&t=general&offset=0&limit=20`（需 `x-zse-96` 签名头）
- 回答详情：`/api/v4/answers/{answer_id}`
- 问题下的回答：`/api/v4/questions/{qid}/answers?limit=20&offset=0&sort_by=default`
- 评论接口：`/api/v4/comments/{answer_id}/root_comments`
- 用户信息：`/api/v4/members/{url_token}`
- 知乎API对频繁请求有严格的限速保护（HTTP 429/403），需要控制请求频率（建议≥2秒/次）
- `x-zse-96` 签名：知乎 API v4 的反爬机制，基于特定加密算法从URL路径计算，版本 3.0.40
- 建议搭配浏览器Cookie（特别是 `d_c0`）使用，以提高API访问稳定性
